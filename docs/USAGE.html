Ollama Runner & Sakana Chat Ratchet - 使い方

概要

Ollamaで動作するローカルLLMとSakana AIを同じ画面上で切り替えながら使えるツールです。

  • Ollama: ローカルで動作するオープンソースLLM
  • Sakana AI: chat.sakana.ai 提供の日本語特化AIチャットサービス

動作要件

  • Ollama がローカルで起動していること（デフォルト: http://localhost:11434）
  • Python 3.8+
  • uv（パッケージ管理）
  • Playwright（ブラウザ自動化用）

起動方法

# 依存関係のインストール
uv sync

# アプリ起動
uv run python app.py

ブラウザで http://localhost:7860 が開きます。

機能説明

モデルタイプ選択

画面上部のラジオボタンで2つのモードを切り替え:

モード 説明
Ollama ローカルのOllamaモデルを実行
Sakana AI chat.sakana.aiにプロンプトを自動送信

Ollamaモード

モデル選択

  • ドロップダウンからOllamaにインストール済みのモデルを選択
  • 🔄 ボタンでモデル一覧を更新

サンプル選択

サンプル 説明
フリートーク 空のプロンプトで自由に入力
要約 日本語テキストを200文字以内で要約
質問応答 ユーザーからの質問に対して回答
メール下書き ビジネスメールの下書きを作成
メッセージ下書き メッセージの下書きを作成

パラメータ

  • temperature: 生成の多様性（0.0〜2.0）
  • top_p: トークン選択の範囲（0.0〜1.0）
  • max_new_tokens: 生成する最大トークン数

ボタン

  • 実行: 選択したモデルでプロンプトを送信
  • キャンセル: 実行中のリクエストを中断
  • クリア: プロンプトと出力をクリア
  • ダウンロード: 完全出力をテキストファイルで保存
  • 履歴を更新: 会話履歴を表示

Sakana AIモード

「Sakana AI」モードを選択すると:

  1. 入力したプロンプトが自動的にChromiumブラウザでchat.sakana.aiに送信されます
  2. 新しいウィンドウでSakana Chatが開き、プロンプトが自動入力されます
  3. プロンプトは履歴に保存されます

注意

  • Sakana Chatは日本国内からのみアクセス可能です
  • ブラウザウィンドウは開いたままになり、応答を確認できます
  • エラーが発生した場合は手動でアクセスしてください

履歴機能

「履歴に保存」チェックボックスをONにすると、会話を保存できます。

「履歴を更新」ボタンで過去の会話履歴を表示できます。

ダウンロード機能

  • 対象: 完全出力欄
  • ファイル名: output.txt（固定）
  • 文字コード: UTF-8
  • 制約: 完全出力欄が空の場合はダウンロードされません

環境変数

変数名 デフォルト値 説明
OLLAMA_BASE_URL http://localhost:11434 Ollama APIのエンドポイント
